# noveltree_yw7

The [noveltree](https://peter88213.github.io/noveltree/) Python program helps authors organize novels.  

*noveltree_yw7* is a plugin providing *.yw7* file import and export. 

![Screenshot](Screenshots/screen01.png)

## Requirements

- [noveltree](https://peter88213.github.io/noveltree/) version 1.0+

## Download and install

[Download the latest release (version 1.0.0)](https://github.com/peter88213/noveltree_yw7/raw/main/dist/nv_yw7_v1.0.0.zip)

- Extract the "nv_yw7_v1.0.0" folder from the downloaded zipfile "nv_yw7_v1.0.0.zip".
- Move into this new folder and launch **setup.pyw**. This installs the plugin for the local user.
